Taro Fetch polyfill

Taro 3+ 非官方 W3C fetch() API 补丁

用法

❗因小程序锁定了全局对象，无法采用标准的 polyfill 方式，只能用 ponyfill 来导入。

import { fetch } from 'taro-fetch-polyfill';

fetch('https://api.github.com')
    .then(response => response.json())
    .then(console.log);
